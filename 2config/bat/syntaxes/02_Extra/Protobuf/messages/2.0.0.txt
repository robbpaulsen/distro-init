* Added a syntax for protobuffer serialized as text eg:

```

health: 100

  name: "big sword"
  damage: 10
}
```

* Renamed a few scope to better match with https://www.sublimetext.com/docs/3/scope_naming.html
* Please report issues at https://github.com/VcamX/protobuf-syntax-highlighting/issues
