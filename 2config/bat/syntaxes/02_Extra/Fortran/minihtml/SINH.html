SINH — Hyperbolic sine function

Description

SINH(X) computes the hyperbolic sine of X.

Syntax

RESULT = SINH(X)

Arguments

X The type shall be REAL or COMPLEX.

Return value

The return value has same type and kind as X.

Example


program test_sinh
  real(8) :: x = - 1.0_8
  x = sinh(x)
end program test_sinh

Specific names
