CHAR — Character conversion function

Description

CHAR(I [, KIND]) returns the character represented by the integer I.

Syntax

RESULT = CHAR(I [, KIND])

Arguments

I The type shall be INTEGER.
KIND (Optional) An INTEGER initialization expression indicating the kind parameter of the result.

Return value

The return value is of type CHARACTER(1)

Example


program test_char
    integer :: i = 74
    character(1) :: c
    c = char(i)
    print *, i, c ! returns 'J'
end program test_char

Specific names

Name Argument Return type Standard
CHAR(I) INTEGER I CHARACTER(LEN=1) F77 and later

Notes

See ICHAR for a discussion of converting between numerical values and formatted string representations.

Standard

Fortran 77 and later

Class

Elemental function
